# ayah-sfdc

Add "Are you a human?" to visualforce pages

## Usage

Initialize an **Ayah** object with your `publisher key` and your `scoring key`

```java
Ayah myAyah = new Ayah('mypublisherkey', 'myscoringkey');
```
